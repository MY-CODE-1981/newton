# 20260226
  新規作成                                                                                                                                      
   
  1. slip_model/ablation.py — ablation実験スクリプト                                                                                            
  2. kiroku/20260226_slip_detection_ablation_study.md — 記録

  ablation.py の構成

  - CONDITIONS dict: A/B/C/D の4条件定義
  - make_config(): 基本Configから各条件のConfigを生成（dataclasses.replace使用）
  - checkpoint_path(): 条件Aは既存パス、B/C/Dは ablation_{id}/ 以下
  - run_ablation(): メインロジック
    - 条件A: 既存checkpoint → evaluate() のみ
    - 条件B/C/D: train() → evaluate() を順次実行
    - 全結果の比較表コンソール出力
    - slip_model/logs/ablation_results.json に保存
  - main(): argparse（--device, --epochs, --skip-train）

  動作確認

  Import と Config 生成を検証済み:
  A: cameras=4cam, joint=True,  in_ch=20
  B: cameras=4cam, joint=False, in_ch=20
  C: cameras=2cam, joint=True,  in_ch=10
  D: cameras=2cam, joint=False, in_ch=10

  実行方法

  cd AS-RTI-acados
  python -m slip_model.ablation                    # フル実行
  python -m slip_model.ablation --epochs 30        # epoch数指定
  python -m slip_model.ablation --skip-train       # eval only（要既存ckpt）
